Computer Graphics

Cartoonify

DESCRIPTION: Given a headshot photo of an individual (e.g. the headshot of Obama), generate the cartoon version of the photo by sketching it using Bézier curves.

TOOLS: C++, OpenGL

Geometric Modeling of a Scene

DESCRIPTION: Replicate a real world scene using hierarchical modeling of the objects. Begin by building prisms from planes using 3D translations and rotations. These prisms are then used to form parts of objects.

TOOLS: C++, OpenGL

Shading

DESCRIPTION: Given a scene, our goal is to generate objects using hierarchical modeling (based on assignment 2) and add shading (colors) to these objects by defining a light source (a 3-dimensional vector), computing the surface normals, and generating the observed colors using Gouraud shading.

TOOLS: C++, OpenGL
